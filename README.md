# README

This is a simple web app to allow employees to punch in and punch out.

Application is created with Ruby on Rails (basic MVC)

## Heroku link
https://lit-lowlands-98746.herokuapp.com/

## System dependencies
** ruby 2.5.1
** rails 5.1
** postgresql@9.6

* run `bundle`
* run `rake db:create && rake db:migrate`

## To run test

* run `rspec`
